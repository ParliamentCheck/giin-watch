# 未照合 speaker_name 判定リスト

speeches テーブルに発言記録があるが、member_id が紐付いていない人物の一覧。
元議員であれば登録すると発言データが活きる。

## 記入方法

判定欄に以下のいずれかを入力してください：

```
元議員                        ← 元議員（院・政党は自動補完を試みる）
元議員 衆議院 立憲民主党      ← 院・政党を指定
元議員 参議院 自民党 3期      ← 期数まで指定
非議員                        ← 官僚・学者・参考人など（登録しない）
スキップ                      ← 不明・保留
```

### 入力例

```markdown
| 枝野幸男    | 61 | 元議員 衆議院 立憲民主党 25期 |
| 植田和男    |  9 | 非議員                       |
| 山田好孝    | 38 |                              |  ← 空欄はスキップ扱い
```
